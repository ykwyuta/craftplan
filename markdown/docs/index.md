# Documentation

Welcome to the Craftplan documentation. These guides cover every major
feature of the platform, from initial setup to day-to-day operations.

## Getting Started

New to Craftplan? Follow the [Self-Hosting guide](/craftplan/docs/self-hosting/) to
get Craftplan running with Docker, then read the [Overview & Planner](/craftplan/docs/overview/) guide
to learn how the main workspace is organized.
Contributing? See the [Development Setup](/craftplan/docs/getting-started/) for the Elixir dev environment.

## Feature Guides

* [Overview & Planner](/craftplan/docs/overview/) — The landing page with schedule, make sheet, and materials tabs
* [Catalog & BOMs](/craftplan/docs/catalog/) — Products, BOM editor, versioning, labor steps, and pricing
* [Inventory](/craftplan/docs/inventory/) — Materials, lots, stock movements, allergens, and forecasting
* [Orders](/craftplan/docs/orders/) — Order lifecycle, statuses, calendar view, and invoicing
* [Purchasing](/craftplan/docs/purchasing/) — Suppliers, purchase orders, and receiving
* [Production Batching](/craftplan/docs/production-batching/) — Batch workflow from allocation to completion
* [Customers](/craftplan/docs/customers/) — Customer database, addresses, and order history
* [Settings](/craftplan/docs/settings/) — General configuration, markup, allergens, CSV import/export
